# TrackYourSats — Support site

A single static page (`index.html`) hosting the TrackYourSats support page,
served via GitHub Pages at `https://ondrejstarek.cz/trackyoursats-support/`.

Used as the **Support URL** in App Store Connect.

- `.nojekyll` tells GitHub Pages to serve the files as-is (skip Jekyll).
- The page is fully self-contained — inline CSS, no external assets, light/dark
  aware, mobile-responsive.
- Sibling repo: [trackyoursats-privacy-policy](https://github.com/ondrastarek/trackyoursats-privacy-policy)
  hosts the privacy policy the page links to.
